1. Root Directory Structure
The project lives at D:\Projects\Personal_Projects\Websites\fitosys. Every file and folder below is relative to this root.
2. app/ Directory — Pages and API Routes
All routes are collocated under app/. Pages use page.tsx. API routes use route.ts. Layouts use layout.tsx.
3. lib/ Directory — Business Logic and API Clients
lib/ contains all non-UI logic. Nothing in lib/ should import from app/ or components/. Every external service has its own subfolder.
4. components/ Directory — UI Components
All components are in TypeScript. No default exports except for page.tsx files. Every component has a single responsibility.
5. types/ Directory — TypeScript Interfaces
All DB table interfaces live in types/database.ts. All API request/response shapes live in types/api.ts. Never use 'any'.
