No GSAP Rule
No GSAP on content or sections. CSS transitions only (0.2s ease). Framer Motion only if scroll-triggered reveals are explicitly required on a specific section. No scroll-jacking. No page loaders. Performance over visual novelty.

NO DEVIATIONS. Two fonts only. If any other font appears, remove it. Barlow Condensed (var(--fd)): - Use for: ALL headlines, stat numbers, price figures, CTAs, feature numbers - Weights: 400, 500 ONLY. Hard cap at 500. Urbanist (var(--fb)): - Use for: ALL body copy, labels, nav, forms, descriptions, testimonials - Weights: 300, 400, 500 for body. 700 ONLY for CTA buttons.
